#MITE
is a project that will help to manage solid municipal waste (MSW) removal more efficiently

We have developed an IoT device for monitoring the fullness of garbage cans. The device is easily and reliably attached to any trash can using 4 screws and with the help of an ultrasonic sensor determines the level of waste in the tank. The operator at any time can see the trash cans on the map and the amount of trash they contain. A GPS version is also available which will allow the operator to track the bins in real time.
